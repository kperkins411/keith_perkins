Classes

CPSC 471 - Applied Artificial Intelligence Introduces applied artificial intelligence through modern deep-learning workflows and tools. Students begin with the mathematics of neural networks and implement backpropagation and multilayer perceptrons using NumPy and PyTorch for regression and classification tasks. The course then focuses on computer vision, including CNN-based image classification, transfer learning with pretrained models from the timm ecosystem, and YOLO-based object detection with metrics such as IoU and mAP. The final portion covers transformers, embeddings, Hugging Face workflows, LLM prompting and fine-tuning, and retrieval-augmented generation systems using vector databases and evaluation techniques.
DATA 301 - Data Science Methodology Introduces the practical methodology of modern data science, guiding students through the full analytical workflow from data preparation and exploratory analysis to modeling and interpretation. Students learn machine-learning methods including K-Means clustering, decision trees, random forests, gradient-boosted trees, and neural networks, along with techniques for evaluating predictive performance and explaining model behavior. The course also covers time-series analysis and applies these methods to real datasets so students develop complete, reproducible data-science workflows.
PCSE 595 - AI Infrastructure short course (Special topics) A two-week short course that teaches students how to run deep-learning and computer-vision experiments using cloud-based GPU infrastructure, with an emphasis on building reproducible remote development environments. In the first week, students learn to provision cloud machines, automate environment setup with startup scripts, and manage development workflows using Linux tools, shell configuration, and scripting. In the second week, they apply this infrastructure to train and fine-tune computer-vision models in a Kaggle-style workflow, covering dataset acquisition, dataloaders, model training, evaluation, GPU monitoring, and competition submission.
CPSC 410 - Operating Systems Examines operating system fundamentals including processes, scheduling, interrupts, and virtual memory management. The course strongly emphasizes multithreaded programming, covering thread creation, race conditions, critical sections, atomics, mutexes, and techniques for safely synchronizing concurrent execution. Students implement and analyze various process schedulers, threaded systems using condition variables, semaphores, and classic concurrency problems such as producer–consumer and deadlock scenarios.
CPSC 327 - C++ programming Introduces C++ programming and development in a Linux environment, including IDE setup, compilation, and use of Git. Students learn core language features such as data types, functions, pointers, memory management, file I/O, and use of the C++ standard library. The course then covers object-oriented programming concepts including classes, inheritance, composition, polymorphism, smart pointers, and basic software design practices.
